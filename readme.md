### Project setup

#### Requires
- Git
- Python3
- Curl or Wget

- **via curl**

```shell
curl -fsSL https://raw.githubusercontent.com/mosesokemwa/super-duper-octo-testing/master/setup | bash
```

- **via wget**
```shell
wget -fsSL https://raw.githubusercontent.com/mosesokemwa/super-duper-octo-testing/master/setup | bash
```
